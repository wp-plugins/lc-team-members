Fully responsive and mobile ready meet the team showcase plugin for wordpress.This plugin will display team members using shortcode on your page. 

== Description ==

LC Team Members will display team members of your company using shortcode on your post or page. You can add and manage their image, social profile's links, short description along with their name and position in your company.

= Features: =

* Easy to use
* Fully responsive
* Simple and light weight
* Font Awesome Icons
* Shortcode enabled
* Custom Post Type
* Major Browser Supported

== Installation ==

There are 3 ways to install this plugin:

= 1. The super easy way =
1. In your Admin, go to menu Plugins > Add
1. Search for LC Testimonial Slider
1. Click to install
1. Activate the plugin
1. A new menu named `LC Team Members` will appear in your Admin

= 2. The easy way =
1. Download the plugin (.zip file) on the right column of this page
1. In your Admin, go to menu Plugins > Add
1. Select the tab "Upload"
1. Upload the .zip file you just downloaded
1. Activate the plugin
1. A new menu named `LC Team Members` will appear in your Admin

= 3. The old way (FTP) =
1. Upload `lc-team-members` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. A new menu named `LC Team Members` will appear in your Admin

== Usage ==

* To insert the Testimonial Members, paste the below shortcode in the content section of your WordPress post/page:

`[lc_team_members]`
